
Here is a Jenkins pipeline script that you can use to execute the Terraform script in an automated way:
```
pipeline {
    agent any

    stages {
        stage('Init') {
            steps {
                // Initialize Terraform
                sh 'terraform init'
            }
        }
        stage('Validate') {
            steps {
                // Validate the Terraform configuration
                sh 'terraform validate'
            }
        }
        stage('Apply') {
            steps {
                // Apply the Terraform changes
                sh 'terraform apply --auto-approve'
            }
        }
    }
}
```
This pipeline script will run the `terraform init`, `terraform validate`, and `terraform apply` commands in sequence. The `terraform apply` command will use the `--auto-approve` flag to automatically approve any changes that need to be made.

Note that you will need to replace the `ami` and `instance_type` variables with the appropriate values for your environment. You can find more information about these variables in the Terraform documentation.
