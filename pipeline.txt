
Here is a Jenkins pipeline script that you can use to execute the Terraform script in an automated way:
```
pipeline {
    agent any

    stages {
        stage('Init') {
            steps {
                // Initialize Terraform
                sh 'terraform init'
            }
        }
        stage('Validate') {
            steps {
                // Validate the Terraform configuration
                sh 'terraform validate'
            }
        }
        stage('Apply') {
            steps {
                // Apply the Terraform changes
                sh 'terraform apply --auto-approve'
            }
        }
    }
}
```
This pipeline script will run the following commands in each stage:

1. `terraform init`: Initializes Terraform and creates a new working directory.
2. `terraform validate`: Validates the Terraform configuration and ensures that it is correct and complete.
3. `terraform apply --auto-approve`: Applies the Terraform changes to the AWS infrastructure, using the `auto-approve` flag to automatically approve any changes that are required.

You can save this script as a file (e.g., `terraform-pipeline.groovy`) and then use Jenkins to run the script and execute the Terraform commands.
