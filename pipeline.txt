
Here is a sample Jenkins pipeline script that you can use to execute the Terraform script in an automated way:
```
pipeline {
  agent any

  stages {
    stage('Init') {
      steps {
        // Initialize Terraform
        sh 'terraform init'
      }
    }

    stage('Validate') {
      steps {
        // Validate the Terraform configuration
        sh 'terraform validate'
      }
    }

    stage('Apply') {
      steps {
        // Apply the Terraform changes
        sh 'terraform apply --auto-approve'
      }
    }
  }
}
```
This pipeline script has three stages: `Init`, `Validate`, and `Apply`. In the `Init` stage, the `terraform init` command is run to initialize Terraform. In the `Validate` stage, the `terraform validate` command is run to validate the Terraform configuration. Finally, in the `Apply` stage, the `terraform apply --auto-approve` command is run to apply the changes.

Note that you'll need to replace the `sh` commands with the appropriate shell command for your environment.
